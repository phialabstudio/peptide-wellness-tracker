Smart automatic reminders that notify you when it's time for your peptide injection.


* 🔔 Get a pop-up reminder on your screen
* 📱 Get a phone notification (if you allow it)

How to Set Up a Shot Reminder
Step 1: Click the "🔔 Reminders" Tab
You'll see a section that says "Shot Reminders"
Step 2: Click "+ Add Reminder" (light coral button)
A form appears:


Fill in these fields:


1. Label (give your reminder a name)


   * Example: "Morning BPC-157"
   * Example: "Tuesday TB-500"


2. Time (what time should you be reminded?)


   * Example: 7:30 AM
   * Pick the time you want the reminder


3. Peptide (which peptide is this for?)


   * Select from your peptide list
   * Example: BPC-157, TB-500, etc.


4. Repeat (how often should you be reminded?)


Choose ONE:


   * ⭐ Every day - Same time, every single day
   * 📅 Every other day - Skip one day, remind the next
   * 🗓️ Weekly - Same day(s) each week
   * 💼 Weekdays - Monday-Friday only
   * 🎉 Weekends - Saturday-Sunday only


5. Which days? (if you chose "Weekly")


   * Check the specific days (Mon, Tue, Wed, etc.)
   * Only shows if you picked "Weekly"


6. Note (optional - what should the reminder say?)


   * Example: "Check injection site"
   * Example: "Don't forget water!"
   * This appears in your reminder notification
Step 3: Click "Save"
Your reminder is now active! ✅

💾 Backup Your Data
Your data is safe on your device, but backup is important!
How to Backup
1. Go to ⚙️ Settings tab
2. Scroll down to "Data Management"
3. Click "Export Data"
4. A file downloads (like peptide-tracker-backup.json)
5. Save it somewhere safe (Desktop, cloud drive, etc.)
How to Restore on Another Device
1. Move the backup file to your new device
2. Open Peptide Wellness Tracker app
3. Go to ⚙️ Settings tab
4. Click "Import Data"
5. Select your backup file
6. All your data appears! ✅
How to Migrate Between Devices
Example: Moving from phone to computer


1. On phone: Export data
2. Send file to yourself (email, cloud, etc.)
3. On computer: Import that file
4. Done! Everything synced 🎉

❓ Common Questions & Answers
Q: Will my data disappear if I close the browser?
✅ No! Your data is saved and stays forever (until you clear browser data or factory reset device).
Q: Can I use this on my phone AND computer?
✅ Yes! Use Export/Import:


1. Export on Device A
2. Import on Device B
3. All your data syncs!
Q: What if I make a mistake logging something?
✅ Click the entry in the table, edit it, and save! Or delete it and re-log.
Q: Does the app work offline?
✅ Yes! Completely offline. No internet needed.
Q: What if the reminder doesn't work?
Try:


* Make sure the app is open (or check if browser is closed)
* Grant notification permission when asked
* Check your computer/phone volume isn't muted
* Refresh the page
Q: Can I change my mind about units later?
✅ Yes! Just go to Settings and change from lbs to kg (or inches to cm). App updates everything.
Q: Is there a mobile app to download?
❌ No need! This works perfectly in your phone's web browser. Just bookmark it like you would an app.
Q: Can I share reminders with other people?
❌ Each person needs their own copy (with their own data). But you can share the HTML file!

🚨 Troubleshooting
App Won't Open
Try:


1. Make sure it's the .html file
2. Double-click it again
3. Or drag it onto your browser window
4. Or right-click → "Open with" → Choose your browser
Data Disappeared
Try:


1. Check if you cleared browser data (and lost it)
2. If you have a backup, import it
3. Otherwise, unfortunately it's gone (always backup!)
Reminders Not Working
Try:


1. Check if reminders are enabled (✅ icon shows)
2. Grant browser notification permission
3. Make sure you didn't close the app entirely
4. Refresh the page (F5)
5. Check if times are set correctly
Charts Not Showing
Try:


1. Refresh the page (Ctrl+R or Cmd+R)
2. Make sure you have entries logged
3. Give it a moment to load
4. Try a different browser
Import Not Working
Try:


1. Make sure it's the right backup file (.json)
2. Don't rename or edit the file
3. Try exporting again to get fresh file
4. Refresh page before importing
